# expense-tracker-web-app
A full-stack web application that allows users to manage daily expenses efficiently. Users can add, update, delete, and categorize expenses. The app also provides total spending calculations and graphical analysis of expenses.

A full-stack Expense Tracker application built using React, Node.js, Express, and MongoDB.

## 🚀 Features
- Add, delete expenses
- View total spending
- REST API integration
- MongoDB database storage

## 🛠 Tech Stack
Frontend: React.js
Backend: Node.js + Express
Database: MongoDB

## 📦 Installation

1. Clone the repository
2. Backend setup:
   cd backend
   npm install
   node server.js

3. Frontend setup:
   cd frontend
   npm install
   npm start

Server runs on: http://localhost:5000
Frontend runs on: http://localhost:3000
